# 💰 GranaSimples

Aplicação web de controle financeiro desenvolvida com Flask.

## 🚀 Funcionalidades
- Controle de saldo
- Entradas e saídas
- Metas financeiras
- Dashboard com gráficos

## 🛠️ Tecnologias
- Python (Flask)
- PostgreSQL (Supabase)
- HTML, CSS, JavaScript

## 📦 Como rodar
pip install -r requirements.txt
python app.py
